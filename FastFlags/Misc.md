### Replace all Decals with a Test Image 
@.rbx.bloxy
```json
{
    "FFlagDebugTestImageDrawItem": "True"
}
```
### Shows the state of a flag
```json
{
    "FStringDebugShowFlagState": "FLAG_HERE"
}
```
#### e.g
```json
{
    "FStringDebugShowFlagState": "DFIntTaskSchedulerTargetFps, ChannelName"
}
```
